jQuery Plugin ztInputHint

Defaults:

  1. The Plugin ignores all elements except text-input-fields like type="text" and type="password". That is not changeable, not yet.
  2. The Plugin ignores the element if there is an value set (not empty), but will reactivate hint if the value is removed.
  3. The Plugin read the title attribute (per setting changeable).
  4. The Plugin inherits attributes like CSS-Class (per setting changeable), size and tabIndex.
  5. The Plugin will not set Name-Attributes to the hints, so there will not be submit.
  6. Changeable options: { hint: 'myname', sourceAttrib: 'myattrib', hintClass: 'myclass'}.

The simplest way

Place this in the javascript area.
$("form#login input").ztInputHint();

Changing sourceAttrib

Changing sourceAttrib, to automatically pick an other attribute:
$("form#login input").ztInputHint({ sourceAttrib: 'alt' });
Demo:

Simple assigning hints directly

Easy manually assign hints to inputfields:
$("form#login input#username").ztInputHint('Username'); $("form#login input#password").ztInputHint('Password');
Demo:

Use an other class for hints

Bind an other CSS-Class to the hints:
$("form#search input").ztInputHint({ hintClass: 'label' });
